# 🎓 毕设项目亮点速览

## 📌 一句话介绍
**LenseMaster AI** 是一个利用 NIMA 深度学习模型 + 多大模型融合（Zhipu GLM-4V / Qwen VL）的**智能摄影美学评论系统**，提供 9 维度专业分析、交互式反馈、实时数据可视化。

---

## 🌟 核心亮点（5大差异化特性）

### 1️⃣ **多大模型自适应融合**
- ✅ **Zhipu GLM-4V** + **Qwen VL-Max** + **Google Gemini** 三模型支持
- ✅ **故障自动转移**：Qwen 失败 → 自动降级到 Zhipu
- ✅ **消除单点依赖**：确保服务 100% 可用性

### 2️⃣ **本地化 NIMA 审美模型**
- ✅ **不依赖外部 API**：TensorFlow SavedModel 本地部署
- ✅ **Python Flask 微服务**：隔离推理计算，支持水平扩展
- ✅ **结合 NIMA 定量 + AI 定性**：客观分数 + 专业评论

### 3️⃣ **交互式智能分析**
- ✅ **点击区域识别**：用户点击照片 → AI 优先分析该区域
- ✅ **9 维度专业评论**：
  - 构图、光影、色彩 (美学维度)
  - 进阶建议、裁剪方案 (实战维度)
  - 主体识别、参数估计 (智能维度)
  - 多维评分 (数据维度)
- ✅ **实时进度反馈**：异步任务 + 轮询查询

### 4️⃣ **工业级异步架构**
- ✅ `@EnableAsync` 异步任务队列
- ✅ 长任务不阻塞用户请求
- ✅ 支持高并发分析场景
- ✅ 流量限制防护（RateLimiter）

### 5️⃣ **完整的微服务生态**
- ✅ **三层分离架构**：
  - React 前端（实时可视化）
  - Spring Boot 后端（业务逻辑）
  - Python 服务（AI 推理）
- ✅ **Docker Compose 一键部署**：4 个容器自动编排
- ✅ **用户系统 + 社交分享 + 数据分析**

## 🔥 创新技术亮点

| 特性 | 实现方式 | 价值 |
|------|--------|------|
| **多模型融合** | Qwen → Zhipu 自动转移 | 服务高可用 |
| **故障降级** | API 失败 → 随机合理分值 | 用户体验连续 |
| **点击交互** | 位置感知 Prompt 工程 | 差异化竞争力 |
| **向量搜索** | 相似度计算 + Top-5 推荐 | 内容发现 |
| **异步处理** | TaskQueue + 长轮询 | 性能优化 |
| **容器化** | Docker Compose | 部署便利性 |

## 📈 核心业务流程

```
1. 用户上传图像
   ↓
2. 前端：EXIF 提取 + CV 特征计算 + 向量生成
   ↓
3. 后端：创建分析任务（异步处理）
   ↓
4. [并行执行]
   ├─ NIMA 评分服务（Python）
   ├─ 多大模型分析（Zhipu/Qwen）
   └─ 相似作品推荐（向量搜索）
   ↓
5. 结果聚合：NIMA + AI 分析 + 推荐
   ↓
6. 前端实时展示 + 社交分享
```
